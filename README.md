# 📈 Caddie

Plataforma de Inteligência em Investimentos desenvolvida para conectar gestores de carteiras aos seus clientes.

O sistema permite o gerenciamento dinâmico de recomendações de ativos (Ações, FIIs, BDRs), definição de preços-teto e análise de viés de mercado em tempo real através de um painel (dashboard) interativo.

## 🛠 Tecnologias

**Front-end:**
- React + TypeScript
- CSS3 (Vanilla)
- Axios

**Back-end e Banco de Dados:**
- C# (.NET 8)
- Entity Framework Core
- SQL Server
